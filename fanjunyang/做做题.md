# EventLoop   
输出打印顺序

``` 
setTimeout(() => {
    console.log(2);
    Promise.resolve().then(() => {
        console.log(6);
    });
}, 0);
Promise.resolve(3).then((data) => {
    console.log(data);          
    return data + 1;
}).then((data) => {
    console.log(data)                
    setTimeout(() => {
        console.log(data + 1)        
        return data + 1;
    }, 1000)
}).then((data) => {
    console.log(data);                
});
```
